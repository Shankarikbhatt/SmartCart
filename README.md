#SmartCart 🛒
The zero-dependency, ultra-responsive grocery ecosystem for the modern terminal and browser.
SmartCart is a fast, lightweight web application designed to fix how we track household inventory. No massive frameworks, no endless NPM packages, and absolutely no clutter. Just pure, reactive frontend engineering tailored for lightning-fast shopping lists.

⚡ The Quick Pitch
Why use SmartCart? Because tracking what you need shouldn't feel like a chore. Most grocery apps are bloated with ads or require cloud accounts just to remember to buy milk. SmartCart brings it back to basics with a gorgeous, keyboard-and-touch-friendly layout that builds your shopping manifest instantly.

 Key Highlights
 
 Three-State Cycle Logic: Tap once to mark as Needed (Vibrant Red). Tap twice to mark as In Pantry (Clean Strikethrough for easy auditing). Tap again to clear.
 
 Blistering Performance: Built entirely on vanilla web technologies. It loads instantly and executes state changes with zero lag.
 
Complete Ingredient Database: Pre-loaded with 12 distinct culinary and household categories, from basic grains and spices to international staples.

 Dynamic Scaling: Need something hyper-specific? Add your own custom items to any category block on the fly.
 
 One-Click Manifest Export: Instantly parses your active states, compiles a beautiful, department-sorted text list, and copies it to your clipboard for easy texting.
 
 Cyberpunk vs. Clean Minimal: Seamless transition between an immersive dark theme and a high-contrast light theme built for bright grocery store aisles.
 
Ink-Saving Print Mode: Includes a dedicated print stylesheet that automatically hides buttons and UI menus, leaving you with a flawless paper checklist.

 The Tech Blueprint
 
We kept the architecture beautifully simple so anyone can learn from, modify, or fork this repository:

Markup: Semantic HTML5
Styles: CSS Grid, Flexbox, and Native Custom Variables
Typography: Syne (for structural impact) and Space Mono (for data clarity)
Logic: Pure, framework-free Vanilla ES6+ JavaScript
